TCG Order → CSV

Paste an unformatted order or card list from TCGPlayer (or similar) and convert it to a clean CSV ready for Moxfield, Archidekt, Deckbox, and more.

Supported formats (one card per line, or two-line blocks):

  • 1x Black Lotus [Limited Edition Alpha] Near Mint
  • 1x Llanowar Elves - Dominaria - Near Mint Foil
  • Black Lotus [Limited Edition Alpha]
  • Two-line: card name on one line, set name on the next

Supported: .mht, .mhtml, .html, .htm — you can upload multiple files.

Set abbreviations powered by the Scryfall API. For Magic: The Gathering cards.
